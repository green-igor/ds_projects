# Проект для интернет-магазина «Викишоп»

К нам обратился интернет-магазин «Викишоп», который запускает новый сервис для того, чтобы пользователи могли редактировать и дополнять описания товаров, как в вики-сообществах.

Заказчику был нужен инструмент, который бы искал токсичные комментарии и отправлял их на модерацию. Этим инструментом стала модель, которая была получена в данном проекте. Значение f1 - метрики полученной модели преодалело 0.94, а сама модель позволила ускорить модерацию комментариев.
