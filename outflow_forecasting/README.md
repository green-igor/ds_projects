# Проект для прогнозирования оттока клиентов телеком компании

Оператор связи «ТелеДом» хочет бороться с оттоком клиентов и для этого планирует предлагать промокоды и специальные условия всем, кто планирует отказаться от услуг связи.

Для того, чтобы определить клиентов, которым будут предлагаться специальные условия было решено разработать прогнозную модель.
В результате исследования была разработана модель, roc-auc которой на тестовой выборке превысила значение 0.93 и позволила заранее предсказать, собирается ли клиент уходить и стоит ли его удерживать.
